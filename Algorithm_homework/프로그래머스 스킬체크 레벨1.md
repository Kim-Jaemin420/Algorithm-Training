# 프로그래머스 스킬체크 레벨 1  

## 예산  
```js
function solution(d, budget) {
  d.sort((a, b) => a - b);
  while (d.reduce((a, b) => (a + b), 0) > budget) d.pop();

  return d.length;
}
```
정렬된 d 배열을 누적합산한 값이 예산보다 크다면, 합이 예산보다 작아질때까지 pop한다.  
그리고, d 배열의 길이를 반환한다.  


## 수박수박수박수박수박수?

```js
function solution(n) {
   let answer = '';
  for (let i = 0; i < n; i++) {
    i % 2 ? (answer += '박') : (answer += '수');
  }
  return answer;
}
```
